An **intelligent Pentecostal church assistant bot** built in Python for Nigerian Pentecostal ministries.  
It handles **Bible Q&A**, **prayer requests**, **counseling**, **testimonies**, and has an **admin dashboard** with learning memory.

---

## Features

- **Bible Q&A** – Answer user questions with scripture references.
- **AI Counseling** – Provide godly, Bible-based advice.
- **Prayer Requests** – Collect and manage prayer requests.
- **Testimonies** – Members can share their testimonies.
- **Admin Panel** – Dashboard, broadcast messages, and export church data.
- **Learning Module** – Bot remembers previous interactions to improve responses.
- **Member Registration** – Collect member details.
